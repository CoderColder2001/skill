# code-plan rehearsals

This directory stores local rehearsal projects used while developing the `code-plan` skill.

## Included rehearsal targets

- `demo-target`: bugfix-focused pre-coding review rehearsal
- `demo-target-sync`: new-module pre-coding review rehearsal

Each rehearsal target contains a small source tree plus generated `code-plan-docs/` output so the skill can be checked against realistic project context.

These rehearsal assets are intentionally kept outside the `code-plan/` skill directory so they do not get packaged into the distributable `.skill` archive.
